# E-Commerce Sales & Customer Behavior Analysis

## Project Overview

This project analyzes an e-commerce dataset to understand sales performance, customer behavior, and product trends using Python (Pandas, Matplotlib).

## Tools Used
  * Python
  * Pandas
  * Matplotlib
  * Jupyter Notebook

## Dataset
The dataset contains 500 records with information on:
  * Orders
  * Customers
  * Products
  * Revenue
  *	Cities
  * Payment methods


## Total Revenue: ₦33,481,644
  * Total Quantity Sold: 1,221 units
  * Top Product: Phone
	* Top Category: Home Appliances
	*	Top City: Kano
	* Best Month: January
	* Highest Spending Gender: Female
	* Most Used Payment Method: Card

## Key Findings
  * Phones generate the highest revenue among all products
  * Home Appliances outperform Electronics in total revenue
  *	Kano is the highest revenue-generating city
	* Sales peak in January
	*	Female customers spend more on average
	* Card is the most preferred payment method
